# How to Find Your API Endpoints on RapidAPI

## Step 1: Go to Your RapidAPI Dashboard

1. Go to https://rapidapi.com/developer/dashboard
2. Log in to your account
3. Click on "My Apps" or "Subscriptions"

## Step 2: Find Each API

### For "Tiktok Data Api":
1. Find "Tiktok Data Api" in your subscriptions
2. Click on it to open the API page
3. Look for:
   - **Base URL** or **Endpoint** (e.g., `https://tiktok-data-api.p.rapidapi.com`)
   - **API Host** (e.g., `tiktok-data-api.p.rapidapi.com`)
   - **Example Request** or **Code Snippets** section
4. Copy the base URL and hostname

### For "Instagram Scraper Stable API":
1. Find "Instagram Scraper Stable API" in your subscriptions
2. Click on it to open the API page
3. Look for:
   - **Base URL** or **Endpoint**
   - **API Host**
   - **Example Request** showing how to get post info
4. Copy the base URL and hostname

### For "API on Twitter":
1. Find "API on Twitter" in your subscriptions
2. Click on it to open the API page
3. Look for:
   - **Base URL** or **Endpoint**
   - **API Host**
   - **Example Request** showing how to get tweet metrics
4. Copy the base URL and hostname

## Step 3: Check the API Documentation

For each API, look at:
- **Request Format**: GET or POST?
- **Required Headers**: What headers are needed?
- **Request Body/Parameters**: What parameters are required?
- **Response Format**: What does the response look like?

## Step 4: Test the API

Use the "Test Endpoint" feature on RapidAPI to:
1. See the exact request format
2. Test with a real URL
3. See the response structure
4. Copy the working code snippet
